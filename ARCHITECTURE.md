## 🚀 使用指南

### 导入模块
```typescript
// 使用统一导入
import { usePetStatus, usePetInteraction } from '@/hooks';
import { PetModel, PetBubble } from '@/components';
import { useBubbleService } from '@/services';
```

### 添加新功能
1. 在对应的功能目录下创建文件
2. 遵循现有的命名规范
3. 更新对应的 index.ts 导出
4. 添加必要的类型定义

### 配置修改
1. 编辑 `config/` 目录下的配置文件
2. 重启开发服务器生效
3. 参考配置文档了解选项

## 🔄 维护建议
