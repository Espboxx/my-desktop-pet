## 设计建议

### 风格特点
- 可爱、友好的卡通风格
- 圆润的形状，适合圆形容器显示
- 明亮温暖的色调（#ffcc80 主色调，#e65100 边框色）

### 表情设计
- **normal**: 微笑、友好的基础表情
- **happy**: 更加开心，可能有闪亮的眼睛或更大的笑容
- **hungry**: 可以显示食物相关元素或饥饿的表情
- **sleepy**: 半闭的眼睛，可能有睡意的符号
- **sick**: 不舒服的表情，可能有绿色的脸色
- **level5**: 酷炫的表情，可能戴墨镜
- **level10**: 兴奋的表情，可能有星星眼
- **level15**: 英雄般的表情，可能有超级英雄元素

### 注视方向
- 眼睛朝向对应方向
- 保持基本表情的同时调整视线
- 可以添加箭头或方向指示符

## 技术规范
- 尺寸：64x64 像素
- 格式：PNG（支持透明背景）
- 背景：透明
- 居中对齐
