## 设计建议

### 风格特点
- 水滴形状的可爱角色
- 流动、透明的质感
- 蓝色系色调（#90caf9 主色调，#1565c0 边框色）

### 表情设计
- **normal**: 基础水滴形状，清澈透明
- **happy**: 可能有波浪效果或更亮的蓝色
- **hungry**: 可能显示饮料杯或渴望的表情
- **sleepy**: 结冰效果或雪花装饰
- **level5**: 彩虹色彩的水滴
- **level10**: 多重波浪效果

### 水滴特色
- 保持水滴的基本形状
- 可以有不同的透明度和反光效果
- 表情主要通过眼睛和整体色彩变化表现

### 注视方向
- 水滴形状的眼睛朝向对应方向
- 可以有水流或波纹指向效果
- 保持水的流动感

## 技术规范
- 尺寸：64x64 像素
- 格式：PNG（支持透明背景）
- 背景：透明
- 居中对齐
- 可以使用渐变和透明效果
